# NEUROSYNC | Neural Performance Ecosystem

NeuroSync is a tactical cognitive synchronization platform. It harmonizes neural patterns through quantified behavior tracking, immersive atmospheric audio, and bioluminescent bio-telemetry. Built on the **Obsidian Stream** design architecture, it provides a high-performance environment for neural evolution.

## 👁️ System Modules

### SEC_01: Neural Tracker (Mood Selection)
Advanced emotional telemetry capturing real-time synaptic states. Features a **Clinical Neural Assistant** that provides data-driven validation and optimization protocols.
- **Protocol**: Clinical/Tactical
- **Tech**: Neural Link V2 (ESM Modular)

### SEC_02: Focus Engine
Atmospheric audio landscapes designed to induce Alpha-wave synchronization and eliminate cognitive friction. Integrated with real-time biometric session tracking.

### SEC_03: Study Vault (Asset Repository)
Encrypted repository for technical disciplines. Categorized by Class (9-12) and Subject (Quant, Physics, Bio-Logic). Access is governed by XP-based authorization tiers.

### SEC_04: Rewards & Progression
Neural evolution is quantified through Experience Points (**XP**). Sessions and logs trigger rewards, allowing users to ascend through 7 verified tiers from *Beginner* to *Singularity Status*.

## 🎨 Design System: OBSIDIAN STREAM

NeuroSync adheres to a strict high-performance aesthetic:
- **Palette**: Deep Obsidian (#080808), Graphite (#1E1E20), and **Signal Orange** (#F97316) accents.
- **Geometry**: Sharp 0px-2px borders targeting technical HUD aesthetics.
- **Textures**: Neural micro-grids, scanlines, and high-frequency glow effects.
- **🚫 PURPLE BAN**: Strict prohibition of violet, purple, or indigo tones (CSS/JS enforcement).

## 🛠️ Technical Stack

- **Logic**: Modern ES6+ JavaScript (ESM Architecture).
- **Styling**: Vanilla CSS (Atomic Tokens) + Tailwind CSS (Utility Framework).
- **Backend**: [Supabase](https://supabase.com/) (PostgreSQL, Auth, RLS, Storage Buckets).
- **Cloud**: Cloudflare Workers + Wrangler Deployment Pipeline.

### ⌨️ Development & Deployment

```bash
# Clone the Neural Repository
git clone https://github.com/neurosync/core.git

# Initialize local environment
npm install

# Build & Synchronize Assets
npm run build

# Deploy to Cloudflare Network
npm run deploy
```

NeuroSync utilizes **Row Level Security (RLS)** and cryptographic session tokens. Biometric data is isolated per UID and never leaves the encrypted Supabase channel without verified authorization.
